# 10,000 Global Checkbox

An interactive real-time grid of **10,000 checkboxes**.
Every click is instantly synced across all connected users using WebSockets.

## 📦 Installation

### 1. Clone the repository

```bash
git clone https://github.com/sagarkemble/10-000-Global-Checkbox.git
cd 10-000-Global-Checkbox
```

---

### 2. Install dependencies

```bash
pnpm install
```

or

```bash
npm install
```

---

### 3. Run the app (development)

```bash
pnpm dev
```

---

### 4. Build the project

```bash
pnpm build
```

---

### 5. Start the server

```bash
pnpm start
```
